# Lista de Exercícios: Elaboração de Endpoints Básicos

Lista para criação de endpoints e familiarização com o fluxo web.

---


1. **Endpoint de Saudação:** Crie um endpoint `GET /saudacao` que retorne a string estática `"Olá, bem-vindo à nossa primeira API!"`.
   
2. **Status do Sistema:** Crie um endpoint `GET /status` que retorne `"Servidor Online e Operante"`.
   
3. **Versão da API:** Crie um endpoint `GET /api/versao` que retorne a string `"v1.0.0"`.
   
4. **Mensagem do Dia:** Crie um endpoint `GET /mensagens/inspiracao` que retorne uma frase motivacional fixa de sua escolha.
   
5. **Citação Geek:** Crie um endpoint `GET /mensagens/geek` que retorne a frase estática `"Que a Força esteja com você!"`.
   
6. **Autor do Projeto:** Crie um endpoint `GET /sobre/autor` que retorne o seu nome completo como criador da API.
   
7.  **Saudação Matinal:** Crie um endpoint `GET /mensagens/bom-dia` que retorne a string `"Bom dia, excelente semana de estudos!"`.
    
8.  **Constante Pi:** Crie um endpoint `GET /matematica/pi` que retorne o valor da constante matemática Pi arredondada para duas casas decimais (3.14).
   
9.  **Elevação ao Quadrado:** Crie um endpoint `GET /matematica/quadrado-de-oito` que calcule internamente o valor de 8² e retorne o resultado.
    
10. **Cálculo de Área Fixa:** Crie um endpoint `GET /matematica/area-quadrado` que calcule internamente a área de um quadrado de lado 15 e retorne o resultado.
    
11. **Operação Composta:** Crie um endpoint `GET /matematica/expressao` que calcule a expressão matemática `(10 + 5) * 2` e retorne o resultado.

14. **Dado Virtual:** Crie um endpoint `GET /jogos/dado` que gere internamente um número aleatório entre 1 e 6 e retorne o valor.
    
15. **Cara ou Coroa:** Crie um endpoint `GET /jogos/moeda` que escolha aleatoriamente e retorne a string `"Cara"` ou `"Coroa"`.
    
16. **Número da Sorte:** Crie um endpoint `GET /jogos/numero-sorte` que gere e retorne um número aleatório entre 1 e 100.
    
17. **Gerador de Senha Básica:** Crie um endpoint `GET /seguranca/senha-aleatoria` que retorne um número aleatório de 4 dígitos (ex: 4921) para simular um PIN.
    
18. **Fruta Aleatória:** Crie um endpoint `GET /aleatorio/fruta` que tenha uma lista interna de 5 frutas e retorne apenas uma escolhida aleatoriamente.

19. **Booleano Aleatório:** Crie um endpoint `GET /aleatorio/verdadeiro-falso` que gere internamente e retorne um valor `true` ou `false` de forma aleatória.

20. **Data Atual:** Crie um endpoint `GET /relogio/data` que capture a data atual do sistema (servidor) e retorne no formato "DD/MM/AAAA".

21. **Hora Atual:** Crie um endpoint `GET /relogio/hora` que capture a hora atual do sistema e retorne no formato "HH:MM".

22. **Ano Atual:** Crie um endpoint `GET /relogio/ano` que retorne apenas o ano em que estamos, extraído do relógio do servidor.

23. **Mês Atual:** Crie um endpoint `GET /relogio/mes` que capture e retorne o mês atual (por extenso ou em formato numérico) baseado no relógio do servidor.

24. **Vogais:** Crie um endpoint `GET /listas/vogais` que retorne uma lista (array simples) contendo as letras `["A", "E", "I", "O", "U"]`.

25. **Cores do Arco-íris:** Crie um endpoint `GET /curiosidades/arco-iris` que retorne uma lista estática com os nomes das 7 cores do arco-íris.

26. **Transformador de Maiúsculas:** Crie um endpoint `GET /texto/maiusculas` que possua uma variável interna com o texto `"programação web"`, transforme esse texto para letras maiúsculas e retorne o resultado.

27. **Dias da Semana:** Crie um endpoint `GET /listas/dias-semana` que retorne uma lista com os 7 dias da semana.

28. **Tamanho do Texto:** Crie um endpoint `GET /texto/tamanho` que defina a string interna `"Desenvolvimento de APIs"` e retorne apenas a quantidade de caracteres dessa string.

29. **Alfabeto:** Crie um endpoint `GET /listas/alfabeto` que retorne um array com todas as 26 letras do alfabeto.
